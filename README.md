# 💰 Finance Dashboard

A modern finance management web app built with React and Vite, featuring role-based access, analytics, and budget tracking.

---

## 🔗 Live Demo

[https://finance-dashboard-drab-eight.vercel.app/](https://finance-dashboard-drab-eight.vercel.app/)

---


## 🚀 Features

- 🔐 Role-based access (Admin & Viewer)
- 📈 Interactive analytics with charts
- 💰 Transaction management (CRUD for Admin)
- 📊 Category-wise budget tracking
- 🌙 Dark / Light mode support
- 💾 Data persistence using localStorage
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS  
- Recharts  
- React Context API  
- Lucide Icons  

---

## 🧠 Architecture

- Component-based structure for scalability  
- Centralized state using Context API  
- Role-based logic handled at state level  
- Clean UI with reusable custom components  

---

## ⚙️ Setup

```bash
git clone [https://github.com/Ankita-1003/Finance_Dashboard]
cd Finance_Dashboard
npm install
npm run dev
```

---

## 🌐 Build

```bash
npm run build
```

---

## 🧠 Assumptions & Considerations

- **Client-Side Persistence:**  
  This version uses localStorage for data persistence. In a production environment, this would be replaced with a backend API and a database.
- **Mock Data:**  
  The application initializes with mock transactions to provide immediate visual context.
